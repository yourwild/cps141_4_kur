# Prompt Template — How to Ask AI for Help

Use this template whenever you start a new assignment or get stuck.
Copy it, fill it in, and paste it into Claude or ChatGPT.

The goal is to get AI to **teach you**, not just give you answers.

---

## Template — Starting a New Assignment

```
I'm a student in an Introduction to Programming with Python course.
I'm working on an assignment and I want you to help me LEARN, not just give me the answer.

## About Me
- I'm new to programming
- I learn best when things are explained step by step with examples
- If I'm doing something wrong, tell me — don't just fix it silently

## The Assignment
[Paste the assignment description or objectives here]

## What I Already Know
[List the concepts you've already covered — loops, functions, lists, etc.]

## What I'm Trying to Do Right Now
[Describe the specific thing you're working on or stuck on]

## My Current Code (if any)
[Paste your code here, or say "starting from scratch"]

## Course Scope — Keep Answers Within These Boundaries
- All code runs inside a Jupyter Notebook — no command line, no terminal scripts
- Do NOT suggest Flask, Django, or any web frameworks
- Do NOT suggest running Python from the command line or a server
- Datasets are provided for most assignments — do not suggest scraping or building data pipelines
- Exception: REST API assignments require an API key — I'll mention when that applies

## How I Want You to Help
- Explain the concept first before showing me code
- Walk me through your reasoning
- Ask me questions to check my understanding
- If you show me code, explain every line
- Point out anything I could do better or more efficiently
```

## Template — When You're Stuck on a Bug

```
I'm getting an error I can't figure out. Help me understand what's wrong
and why — don't just fix it for me.

## My Code
[paste your code]

## The Error Message
[paste the exact error]

## What I Think Is Happening
[your best guess at what's wrong — even if you're not sure]

## What I've Already Tried
[anything you've already attempted]
```

## Template — When You Want a Code Review

```
I finished this code and I want feedback before I submit it.
Be honest — tell me what's good, what's bad, and what a more
experienced developer would do differently.

## My Code
[paste your code]

## What It's Supposed to Do
[brief description]

## Specific Things I'm Unsure About
[any parts you're not confident in]
```

---

## Tips for Talking to AI

- **Be specific.** "It's not working" gives AI nothing. "I get a KeyError on line 12" is useful.
- **Paste your actual code.** AI can't help with code it can't see.
- **Ask why, not just how.** "Why does this work?" teaches you more than "show me how to do this."
- **Push back.** If an explanation doesn't make sense, say so. Ask for a different explanation or a simpler example.
- **Don't just copy code.** If you can't explain what it does line by line, you don't own it yet.

---

*Remember: The goal isn't to finish the assignment. The goal is to understand it.*
