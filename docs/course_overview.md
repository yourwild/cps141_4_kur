## Programming Environment

**Everything happens in Jupyter Notebooks.**

From your instructor:
- All work is completed in Jupyter Notebooks
- No need to run Python via command line or server
- Python is written and executed interactively within Jupyter
- External scripts are not required and are discouraged for this course

## Topics Covered

| Topic | What You'll Learn |
|-------|------------------|
| Control Structures | Loops (`for`, `while`), conditionals (`if`, `elif`, `else`) |
| Data Structures | Strings, lists, dictionaries and their built-in methods |
| Functions | Writing, calling, and reusing functions |
| Classes | Building objects and using them in programs |
| File Processing | Reading and writing text files and CSVs |
| Data Analysis | Accumulating, counting, and deriving statistics |
| Data Visualization | Charts and graphs that communicate clearly |
| APIs & JSON | Making HTTP requests, parsing JSON responses |
| Python Libraries | pandas, matplotlib, seaborn, plotly, requests |

---

## Course Objectives

By the end of this course you will be able to:

- [ ] Write programs using `for` and `while` loops
- [ ] Write programs using `if` statements in all their forms
- [ ] Work with strings and string functions
- [ ] Work with lists and list functions
- [ ] Work with dictionaries and dictionary functions
- [ ] Develop a class and use its objects in a program
- [ ] Write programs that accumulate and count data
- [ ] Use list comprehensions, `map`, `filter`, and `reduce`
- [ ] Process data from text and CSV files
- [ ] Write and use functions
- [ ] Access web services using REST APIs
- [ ] Parse JSON data returned from a web service
- [ ] Write programs that produce basic statistics
- [ ] Visualize information in meaningful, human-readable ways
- [ ] Use Python libraries to process and visualize data

## What This Course Does NOT Cover

This is a foundations course. The following are explicitly out of scope:

- ❌ Flask, Django, or any Python web framework
- ❌ Running Python via command line or server environments
- ❌ Web deployment or hosting
- ❌ Database design or SQL
- ❌ Advanced software architecture

If you find AI suggesting any of these approaches while working on assignments,
redirect it. Use the scope section in `prompt_template.md` to keep answers relevant.

## Key Takeaways

- All coursework is completed in **Jupyter Notebooks**
- Focus is on **foundational Python programming and data work**
